Easily integrate an n8n-powered chat widget into your WordPress site.

== Description ==

MCOD N8N node chat advanced allows you to connect your WordPress site to n8n workflows via a customizable chat widget.

**Features:**
*   **Customizable Appearance:** Change colors, titles, and welcome messages to match your brand.
*   **Header Customization:** Upload a custom logo and change header styles.

*   **Secure:** Built with WordPress best practices.
*   **Developer Friendly:** Includes hooks and a debug mode for easy troubleshooting.

== Installation ==

1.  Upload the `mcnac-n8n-chat-advanced` folder to the `/wp-content/plugins/` directory.
2.  Activate the plugin through the 'Plugins' menu in WordPress.
3.  Go to **MCOD Chat** to configure your webhook URL and customize the chat widget.
