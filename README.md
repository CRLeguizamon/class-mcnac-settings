## üöÄ Installation

1.  Download the repository or the standard plugin ZIP.
2.  Upload the `mcnac-n8n-chat-advanced` folder to your `/wp-content/plugins/` directory.
3.  Activate the plugin through the **Plugins** menu in WordPress.
4.  Go to **Settings > MCNAC N8N Chat** to configure.

## ‚öôÔ∏è Configuration

1.  **Webhook URL:** Enter your n8n webhook URL (POST method).
    *   *Tip: Ensure your n8n workflow returns a JSON response with a `text` or `output` field.*
2.  **Appearance:**
    *   **Chat Logo:** Upload a 40x40px image for the header.
    *   **Colors:** Pick your brand colors for a seamless look.
3.  **Text:** Customize the welcome message and labels to match your language/tone.
